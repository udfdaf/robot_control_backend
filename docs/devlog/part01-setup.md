# Part 01 – Project Setup & Development Environment

## 목표
- 백엔드 프로젝트를 시작하기 위한 기본 개발 환경 구성
- 코드 작성 및 실행이 가능한 초기 상태 확보

## 진행 내용
- Node.js(LTS) 및 npm 개발 환경 구성
- NestJS 기반 백엔드 프로젝트 초기화
- 로컬 개발 서버 실행 확인
- Swagger를 이용한 API 문서화 환경 설정
- GitHub 저장소 연결 및 초기 커밋 구성

## 문제 및 해결
- 기존에 다른 GitHub 계정 자격 증명이 PC에 저장되어 있어 push 권한 오류 발생
  - Windows 자격 증명 관리자를 통해 기존 인증 정보 삭제 후 재인증하여 해결

## 결과
- NestJS 기반 백엔드 서버가 로컬에서 정상 실행되는 상태 확보
- Swagger를 통한 API 문서 접근 가능
- 기능 개발을 바로 시작할 수 있는 프로젝트 초기 구조 완성
